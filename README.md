##Установка сайта:

**Пункт 1:** Что бы установить проект на ваш сервер нам нужно его скачать. Для этого ***нажмите на зеленную кнопку "Code"***, в высветившемся окне ***нажмите "Download ZIP"***.

**Пункт 2:** Вам нужно поменять фамилию и пароль. На данном этапе разработки сайта Админ-панель не предусмотрена! Так что пока что нам ***надо распаковать архив*** и для изменения ***зайти в VS Code***.

**Дальше попунктно:**
- Откройте файл `index.html`(Он находиться по пути `public/index.html`).
- Далее в строке **6** поменяйте  `<title>` на свои данные.
- Так же в строке **12** так же поменяйте `<h1>`.
- На сайте так же предусмотрен пароль, во имя избежания случайного добавления или удаления работ из вне, для того что бы его поменять, нам нужно:
- Найти папку `api`(Она находится по пути `public/api`).
- После, в каждом файле, кроме `get_sites.php`, `get_works.php`, в строке **4** поменять содержимое переменной `$PASSWORD`.

***Сайт готов для выгрузки на сервер!***

**Пункт 3:** Для выгрузки на сервер **заархивируйте файл в ZIP***, после деуствуйте по пунктам ниже:
- Откройте на beget файловый менеджер.
  <img width="958" height="348" alt="image" src="https://github.com/user-attachments/assets/ebd5fe6f-07ab-4165-ac6b-1253899d65f2" />
- Перейдите в папку вашего домена(Обычно это последняя папка).
- После удалите все содержимое этой папки.
- После нажмите на верхней панели значок ***Загрузить файл***, после этого у вас высветится окно.
<img width="956" height="896" alt="image" src="https://github.com/user-attachments/assets/dd44c402-a704-4850-92ec-f44bc986c6ce" />
- В этом окне нажмите кнопку ***Browse...*** и выберете наш ZIP архив.
- после выгрузки архива на сайт вы можете ***нажать кнопку закрыть*** в модальном окне.
- Дальше вам нужно распаковать архив внутри сервера, для этого сначала в ***правой половине экрана*** нужно открыть эту же папку(Это нужно для того что бы распоковать архив там же).
- После на ***левой части экрана*** нажмите ***ПКМ*** на архив, в выпавшем списке ***нажмите распаковать***,после вам высветиться окно с вводом пути куда распаковать файл(Файл нужно распаковать в папку домена), нажимаем ***ОК***.
- После этого мы можем ***удалить запакованный архив***.
- В ***правой части экрана*** откройте папку распакованного архива.
- Выделите все файлы внутри папки(Shift + ЛКМ по первому файлу и последнему), далее ***нажмите ПКМ*** по выделеным файлам, в выпавшем списке ***выберите переместить*** и нажмите ***ОК***.
- Удалите пустую папку архива.
- Переименуйте папку `public` в `public_html`.
- После этого в папке `public_html` создайте папку `uploads`.
- В итоге у вас должно получиться так.
